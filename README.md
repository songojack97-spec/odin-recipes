# odin-recipes
Description
This is my first project from The Odin Project's Foundations course. The goal of this project is to build a basic recipe website using foundational web development skills

Skills Demonstrated
By completing this project, I will have demonstrated the following skills:
* **Git & GitHub:** Initializing repositories, cloning, staging changes, committing, and pushing code.
* **HTML Basics:** Structuring a webpage using headings, paragraphs, lists, links, and images.
* **Directory Navigation:** Working efficiently with files and folders through the command line.
